## 🔧 如果遇到问题

### 端口被占用
```bash
# 查看占用8080端口的进程
lsof -i :8080

# 杀死进程
kill -9 <PID>
```

### 无法访问
1. 检查防火墙设置
2. 确认服务已启动
3. 尝试使用 `127.0.0.1:8080` 而不是 `localhost:8080`

### 样式显示不正常
1. 清除浏览器缓存 (Cmd+Shift+R 或 Ctrl+Shift+R)
2. 确认使用的是现代浏览器
3. 检查网络连接 (需要加载字体)
