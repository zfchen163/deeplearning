## 💡 学习建议

### 每日学习计划
- **时间**: 每天30-60分钟
- **方法**: 
  1. 先看理论(10-15分钟)
  2. 运行代码(10-15分钟)
  3. 修改参数实验(10-15分钟)
  4. 做笔记总结(5-10分钟)

### 学习技巧
1. **不要跳过基础** - 基础不牢,地动山摇
2. **动手实践** - 看懂≠会用,一定要自己写代码
3. **理解原理** - 不要死记硬背,理解为什么这样做
4. **循序渐进** - 不要着急,每个概念都需要时间消化
5. **坚持学习** - 3个月后你会惊讶于自己的进步

### 遇到问题怎么办?
1. **重新阅读课程** - 很多问题第二次看就懂了
2. **查看相关课程** - 可能需要先学习前置知识
3. **运行代码调试** - 实际操作能加深理解
4. **搜索资料** - Google、Stack Overflow是好朋友
5. **做笔记记录** - 记录问题和解决方法

## 🛠️ 常见问题

### Q: 服务器启动失败?
A: 检查:
- Go是否正确安装 (`go version`)
- 端口8080是否被占用
- 是否在正确的目录

### Q: 看不到课程列表?
A: 检查:
- 是否运行了笔记本优化脚本
- 笔记本文件是否在正确位置(项目根目录)
- 浏览器控制台是否有错误

### Q: 代码运行出错?
A: 
- 确保安装了PyTorch和相关依赖
- 检查Python版本(需要3.8+)
- 查看课程中的环境配置说明

### Q: 如何修改端口?
A:
```bash
PORT=3000 go run main.go
```
