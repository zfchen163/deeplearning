```markdown

```

**现在 ✅:**
```markdown

🤖 让计算机自动算导数,不用手算

## 📝 生活中的例子:自动批改作业

老师批改作业:
- 手动批改 ❌: 一个一个看,很慢
- 自动批改 ✅: 机器自动批,很快

## 🤔 为什么需要自动求导?

问题1: 手算导数太麻烦
- 比如这个函数: f(x) = (x² + 3x) × sin(x)
- 手算: 要用链式法则,很复杂
  - **什么是链式法则?** 就像"一层一层剥洋葱"
  - 这个函数是"复合函数"(函数套函数)
  - 需要一层一层地算导数,很麻烦
- 自动求导: 一行代码搞定!

## 🌍 实际应用场景

1. 训练神经网络
2. 优化算法
3. 物理仿真
...
```
